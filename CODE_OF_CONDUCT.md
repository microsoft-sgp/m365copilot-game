# Code of Conduct

This project has adopted the [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/).

For more information, see the [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) or contact [opencode@microsoft.com](mailto:opencode@microsoft.com) with questions or comments.

Project participants are expected to act with respect, professionalism, and care for the community. Reports of unacceptable behavior should follow the reporting guidance in the Microsoft Open Source Code of Conduct.
